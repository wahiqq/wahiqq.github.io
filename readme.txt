== Description ==

Anchor is another premium WordPress theme for creative and digital agency layouts. With its combination of powerful full screen navigation and fully responsive retina ready pages, it is also an amazing choice for a freelancer portfolio. The typography, as for all Themezinho products, is clean and simple, adding to its appeal.
